# ShomsRadar: Decoding the Fear of Crime in Shomolu

A React-based geospatial visualization tool for analyzing fear of crime predictors in Shomolu and Bariga, Lagos.

## 🚀 Quick Start

1.  **Install Dependencies:**
    ```bash
    npm install
    ```

2.  **Run Locally:**
    ```bash
    npm run dev
    ```
    Open the link shown in the terminal (usually `http://localhost:5173`).

3.  **Build for Production:**
    ```bash
    npm run build
    ```

## 🛠️ Features
*   **Interactive Map:** Visualizes 50 data points with fear levels (Red = High, Green = Low).
*   **Smart Filters:** Toggle "Broken Windows", "Dark Spots", and "Escape Routes" to find hotspots.
*   **Analytics:** Real-time charts for Fear vs. Lighting and Victimization Rates.
